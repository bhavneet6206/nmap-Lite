# 🔍 nmap-Lite

A lightweight multithreaded port scanner built in Python with basic service detection and banner grabbing.

## 🚀 Features

* Fast multithreaded port scanning
* Banner grabbing for service detection
* Clean and readable output
* Custom port range support

## 🛠️ Usage

```bash
python scanner.py -t <target_ip> -p <port_range>
```

Example:

```bash
python scanner.py -t 192.168.1.1 -p 1-1000
```

## 📌 Tech Used

* Python
* Socket Programming
* Threading
